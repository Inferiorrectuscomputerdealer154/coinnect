# Coinnect Roadmap

> Beta — priorities shift based on user feedback. Updated March 2026.

## Now ✅

- **10+ live data sources** — Binance, Kraken, Coinbase, Bitso, Luno, Bitstamp, Gemini, Wise, Buda.com, CoinGecko, Frankfurter (ECB)
- **19 remittance providers** with published fee estimates
- **1,000+ routing edges** across fiat, crypto, and P2P
- **50+ currencies** including Gulf (AED, SAR, QAR), Eastern Europe (TRY, UAH, PLN), West Africa (XOF, XAF)
- **Public REST API** — no signup required for basic use
- **MCP server** for AI agents (3 tools)
- **Telegram bot** — @the\_coinnect\_bot
- **Open data** — daily CSV snapshots, CC-BY 4.0
- **15 languages** with geo-detection
- **Self-serve API keys** — instant, no email

## Next

- **More live-rate providers** — replacing static estimates with real API data
- **Flutterwave integration** — live rates for African corridors (NGN, GHS, KES, ZAR, UGX, TZS)
- **Delivery method filters** — bank deposit, cash pickup, mobile money
- **Deep-links to providers** — "Go to Wise →" with pre-filled corridor
- **Community rate verification** — report real rates, verify with proof
- **SEO pages per corridor** — `/send/usd-to-ngn` for discoverability
- **Embeddable widget** — `<iframe>` for blogs and finance sites
- **Price alerts** — get notified when a corridor drops below your threshold
- **Provider outreach** — partnerships for live rate access

## Later

- **x402 micropayments** — USDC on Base L2, pay-per-query for unlimited API access
- **Machine Tipping Protocol** ([mtp.bot](https://mtp.bot)) — micropayments to rate contributors
- **Stellar anchors** — M-Pesa, Wave, GCash via Stellar network
- **Taproot Assets** — stablecoins over Lightning Network
- **SDKs** — Python and JavaScript client libraries
- **WhatsApp bot** — conversational access for users without browsers
- **USSD/SMS gateway** — for feature phones in Africa

## How to contribute

We especially need:
- Exchange adapters with live APIs
- Rate accuracy testing and verification
- Translations
- Documentation
